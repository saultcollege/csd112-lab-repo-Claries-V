Glossary

Opening tag
Element name inside angle brackets, indicates where elements begins.
Closing tag
Element name inside angle brackets, with forward slash before name; indicates where elements begins.
Content
The content of the element (between the opening and closing tags), may be plain text and contain other elements.
HTML attributes
Contains information about the element beyond the content and can used to identify an element for styling and in scripts.
Favicons
Can be used visually distinguish your site in tabs or bookmarks.
